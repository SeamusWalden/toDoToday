# Install
To get the project setup and the python packages installed run: 

cd to-do-today <br>
npm run base-install <br>
npm run start
 <br>
 
To begin the backend:

Windows: <br>
npm run start-api 

Mac: <br>
npm run start-api-mac

Api can face the CORS issues when run on Eduro
